# Frontend de Healthora

Aplicacion cliente de Healthora construida con React 19, Vite y TypeScript.

## Comandos

```bash
bun install
bun run dev
bun run build
```

## URL local

- `http://localhost:5173`

## Notas

- Consume la API del backend configurada por `VITE_API_URL`.
- Usa Clerk para autenticacion en cliente.
- El catalogo conserva categoria y pagina al volver desde detalle.
- Las imagenes de productos se sirven desde `public/products/`.
